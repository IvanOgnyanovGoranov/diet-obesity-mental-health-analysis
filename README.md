# Obesity and Depression: A Cross-Country Analysis

Exploring whether country-level obesity rates are associated with depression rates,
using publicly available global health data from 1990 to 2019.

## Data
- Mental health prevalence: https://www.kaggle.com/datasets/imtkaggleteam/mental-health
- Obesity rates: https://ourworldindata.org/grapher/share-of-adults-who-are-overweight

## Requirements
pip install pandas matplotlib seaborn

## Usage
Open Project.ipynb in Jupyter and run all cells.

## Finding
No robust association between obesity and depression was found at the global level.
